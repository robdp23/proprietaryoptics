Pipeline Tools

Tools &
Utilities

Custom tools built across productions — open to share on request. Built for Houdini, Deadline, ShotGrid, and general VFX pipeline workflows.

01
Image Comparison
Side-by-side render comparison with three modes: Wipe (drag divider), Onion skin (opacity blend), and Diff (pixel difference heatmap). Runs as a local Python server with Windows Explorer context menu integration — right-click any two images to compare instantly.
PythonHTML CanvasWindowsLocal server
02
Farm Analytics
Generates self-contained HTML/ECharts dashboards from Deadline CSV exports. Includes daily render hours stacked bar chart, per-job breakdown, ShotGrid fallback handling, site filtering, and full Windows/Linux cross-platform support via Rez.
PythonEChartsDeadlineRezShotGrid
03
Storage Analytics
Two-script pipeline: prewash_xls.py cleans TreeSize exports via LibreOffice headless, then the dashboard generator produces interactive HTML breakdowns by department and show. Useful for tracking disk usage across large productions without external dependencies.
PythonEChartsXLSLibreOffice
04
Seq QC Submit
PySide2 tool for sequence-level QC render submission outside Houdini. Programmatically builds LOP networks per shot (pxoShotContext → light rig → caches → camera → Arnold ROP) and submits to Deadline with correct frame ranges and camera assignments from ShotGrid.
PythonPySide2HoudiniShotGridDeadline
